Introduction & Overview

Background

The construction industry in New Zealand plays a vital role in the nation's economy, contributing approximately 7% of GDP and employing over 300,000 people. Despite its significance, the sector faces persistent challenges including cost overruns, schedule delays, and quality concerns.

Recent surveys reveal that 90% of contractors experienced project delays in the past six months, while 83% identified rising costs as a significant challenge. These statistics underscore the critical need for effective project management tools and methodologies.

The Promise of EVM

Earned Value Management (EVM) offers a practical solution to address project uncertainty in the current economic environment. By providing early warning signs of potential schedule delays or budget overruns, EVM equips project managers with the data necessary to make informed adjustments.

Key Challenge

NZTA's new Physical Works Schedule Standard mandates EVM reporting for all capital works projects exceeding $5 million, raising questions about industry readiness.

Research Objectives

  • Assess EVM adoption in NZ construction
  • Identify influencing factors
  • Provide practical recommendations

Core EVM Principles

EVM integrates scope, schedule, and cost parameters to assess project performance objectively through three core principles:

  • Planning: Defining project scope and integrated baseline
  • Measuring Progress: Objective tracking of completed work
  • Performance Analysis: Identifying deviations and trends
üìà

Key Metrics

Essential EVM metrics provide insights into project health:

  • PV (Planned Value): Budgeted cost of scheduled work
  • EV (Earned Value): Budgeted cost of performed work
  • AC (Actual Cost): Actual cost of performed work
  • CPI & SPI: Cost and Schedule Performance Indices

The Proposed EVM Framework

Based on research findings and the infrastructure flyover case study, this framework provides practical guidelines for implementing EVM in New Zealand construction projects.

1

Integration with Existing Systems

Leverage existing Cost Breakdown Structure (CBS) and scheduling software. Map cost codes to schedule activities without requiring specialized EVM software.

Key Actions:

  • Align WBS with CBS framework
  • Map each schedule activity to corresponding cost codes
  • Use familiar tools (P6, MS Project) for implementation
2

Budget Distribution Methodology

Apply appropriate budget allocation based on activity characteristics:

Measurable Activities:

  • Direct relationship between cost and deliverables
  • Distribute budget based on quantity or units

Non-Measurable Activities:

  • Use daily rate approach for time-based activities
  • Account for advance payments and procurement
3

Data Integration Platform

Implement accessible reporting solutions that integrate multiple data sources:

Recommendations:

  • Use Power BI or similar for automated integration
  • Connect financial systems with scheduling databases
  • Create user-friendly dashboards for all stakeholders
4

Monthly Review Process

Establish structured performance review cycles:

Review Components:

  • Update schedule progress and earned value
  • Reconcile actual costs from financial systems
  • Analyze variances and performance indices
  • Validate metrics with construction engineers
  • Forecast completion based on current trends

Case Study: Major Infrastructure Flyover Project

Project Overview

  • Project Type: Major Road Flyover
  • Location: Auckland, New Zealand
  • Duration: March 2024 - October 2025
  • Structure: 600m long, 10m high bridge
  • EVM Implementation: April 2024

Key Implementation Steps

1. System Integration

Successfully integrated EVM with existing CBS (26 cost accounts) and Primavera P6 schedule without specialized software.

2. Cost Mapping

Mapped each P6 activity to corresponding cost codes, enabling accurate budget distribution and performance tracking.

3. Power BI Reporting

Implemented Power BI dashboards integrating JD Edwards financial data with P6 scheduling data for comprehensive reporting.

4. Baseline Correction

Addressed initial CPI discrepancies by incorporating procurement activities, improving metric accuracy.

Lessons Learned

  • Standard construction payment practices (advance payments) must be considered in EVM implementation
  • Democratizing access to performance data enhances organizational adoption
  • Monthly review sessions with construction engineers validate metric accuracy
  • Remaining Work analysis provides valuable forward-looking insights

Conclusion & Recommendations

Research Summary

This research demonstrates that Earned Value Management can be successfully implemented in New Zealand construction projects using existing tools and systems. The infrastructure flyover case study proves that EVM delivers tangible benefits including cost savings, schedule optimization, and enhanced project control.

The proposed framework addresses the gap between EVM theory and practical application, providing clear guidelines tailored to New Zealand's construction environment. By leveraging familiar tools and focusing on integration rather than replacement, organizations can achieve effective EVM implementation without significant disruption.

Key Recommendations

For Project Managers

  • Start with existing CBS and scheduling systems
  • Focus on accurate cost-activity mapping
  • Implement monthly review processes
  • Use accessible reporting platforms

For Organizations

  • Invest in data integration capabilities
  • Provide EVM training for project teams
  • Standardize implementation approaches
  • Support pilot projects for learning

For Industry

  • Develop NZ-specific EVM guidelines
  • Share implementation case studies
  • Collaborate on best practices
  • Advocate for consistent standards

Future Research Directions

Further research is recommended in the following areas:

  • Development of New Zealand-specific EVM standards aligned with local practices
  • Integration of EVM with emerging technologies (AI, machine learning, IoT)
  • Comparative studies across different project types and scales
  • Investigation of EVM application in sustainable construction practices

With NZTA's new requirements and increasing project complexity, EVM adoption is no longer optional but essential for New Zealand's construction industry. This research provides the practical framework needed to make that transition successful.
